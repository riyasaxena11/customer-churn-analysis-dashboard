# **# 📊 Customer Churn Analysis Dashboard**



An end-to-end \*\*Data Analytics Project\*\* that analyzes customer churn using \*\*SQL\*\* and \*\*Power BI\*\*. The project identifies customer retention patterns, analyzes churn drivers, and presents business insights through an interactive dashboard.



\---



## **## 📌 Project Overview**



Customer churn is one of the biggest challenges for subscription-based businesses. This project analyzes customer behavior to identify the factors that influence churn and provides actionable insights to improve customer retention.



The project combines \*\*SQL\*\* for data analysis and \*\*Power BI\*\* for interactive dashboard development.



\---



## **## 🎯 Business Problem**



The company wants to answer the following questions:



\* Which customers are most likely to churn?

\* Which contract type has the highest churn?

\* Does payment method affect customer retention?

\* How do internet services influence churn?

\* Does customer tenure impact churn?

\* Which customer segments require retention strategies?

# **# 📈 Key Business Insights**



\* Overall customer churn rate is \*\*26.58%\*\*.

\* \*\*Month-to-Month\*\* contract customers have the highest churn.

\* \*\*Electronic Check\*\* customers show the highest churn rate.

\* Customers \*\*without Tech Support\*\* are significantly more likely to churn.

\* \*\*Fiber Optic\*\* customers have higher churn compared to DSL users.

\* Customers with shorter tenure have a greater risk of leaving.
